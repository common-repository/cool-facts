Cool Facts WordPress Plugin

== Description ==
The Cool Facts Plugin lets you show off a random cool fact via shortcode or widget.

= Shortcode =
The Shortcode will show a random fact on every page load.

[coolfacts]

= Widget =
Show off a random fact in a widget area with a click to reload fact via AJAX.

== Installation ==
1. Upload the cool-facts directory to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
